# HR Attrition Analysis Dashboard 📊

## Overview
An interactive Excel dashboard analyzing employee attrition 
using IBM HR Analytics dataset from Kaggle (1,470 employees, 
35 variables).

## Key Findings
- Overall attrition rate: 16.1% (237 out of 1,470 employees)
- Sales department has highest attrition rate
- Employees aged 25-34 show highest attrition
- Low salary band (Under 3K) drives highest attrition
- Poor work-life balance linked to higher attrition

## Features
- 3 KPI Cards (Total Employees, Total Attrition, Attrition Rate)
- 4 Interactive Charts
- 5 Slicers for dynamic filtering
- Dark navy professional theme

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## Data Source
IBM HR Analytics Dataset — Kaggle
